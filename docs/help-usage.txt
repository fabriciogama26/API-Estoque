Guia rapido do painel de ajuda
------------------------------

- Onde ficam os textos
  - Cada pagina tem o seu JSON em `src/help/` (ex.: `helpSaidas.json`, `helpMateriais.json`, `helpPessoas.json` etc.).
  - Alguns arquivos trazem o topico aninhado (ex.: `{ "saidas": { ... } }`); o HelpButton trata ambos os formatos.
  - Campos aceitos: `title`, `summary`, `steps` (title/description e media/mediaHint/mediaCaption/mediaAlt), `notes`, `links`. Opcionalmente `sections` (title/items), que o componente converte em steps.
  - O componente HelpButton le diretamente o arquivo do topico; nao existe mais `helpContent.json`.

- Como adicionar ou trocar imagens
  1) Salve o arquivo em `public/help/<topico>/nome-da-imagem.webp` (prefira .webp ou .jpg leve).
  2) No passo correspondente:
     - `media`: "/help/<topico>/nome-da-imagem.webp"
     - `mediaAlt`: "Descricao curta da imagem"
     - `mediaCaption`: "Opcional, contexto extra"
  3) Se ainda nao tiver imagem, use apenas `mediaHint` (texto de placeholder).
  4) URLs externas tambem funcionam em `media`, mas guardar em /public evita depender de rede.

- Como criar um novo topico/pagina
  1) Crie um JSON em `src/help/help<Nome>.json` (ex.: `helpSaidas.json`) com a estrutura completa (`title`, `summary`, `steps/sections`, `notes`).
  2) Se preferir aninhar, use `{ "<topico>": { ... } }` â€” o HelpButton aceita os dois formatos.
  3) Importe e use `<HelpButton topic="suaChave" />` no cabecalho da pagina (prop `actions` do `PageHeader`).
  4) Adicione imagens em `/public/help/suaChave/` se quiser ilustracao.

- Como editar a ajuda do campo Agente (Cadastro de Acidentes)
  - Arquivo: `src/help/helpAcidentesAgentes.json`.
  - Campos: titulo, resumo e lista de agentes (cada um com descricao e tipos/lesoes).
  - O icone ao lado de "Agente *" abre esse modal especifico; ajuste o texto no JSON para refletir o uso interno da empresa.

- Dicas de formato
  - Mantenha texto ASCII simples para evitar problemas de codificacao.
  - `mediaHint` e `mediaCaption` sao leves e nao exigem arquivo.
  - Limite passos a conteudo pratico (o modal ja suporta lista, notas e links).

- Ajuste recente (Configuracoes)
  - Apenas master/admin acessam a pagina; use `helpConfiguracoes.json` para refletir roles/permissoes/historico (antes/depois e quem alterou).
